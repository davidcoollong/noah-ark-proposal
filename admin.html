🔑 GitHub 토큰 발급 방법

GitHub에 데이터를 저장하려면 1회 발급 후 아래 입력란에 붙여넣기 하세요.

1
GitHub 로그인 후 토큰 페이지 열기
아래 버튼을 클릭하면 GitHub 토큰 발급 페이지가 열립니다.
(GitHub에 로그인 되어 있어야 합니다)
2
토큰 기본 설정
• Token name: noah-ark-admin 입력
• Expiration: No expiration 선택 (만료 없음)
• Repository access: Only select repositories 선택
  → noah-ark-proposal 저장소 선택
3
권한 설정 (가장 중요)
Repository permissions 섹션에서:
• Contents → Read and write 선택
나머지는 기본값(No access) 그대로 두세요.
4
토큰 생성 및 복사
페이지 하단 Generate token 버튼 클릭
→ 생성된 github_pat_... 토큰 전체 복사
⚠ 이 페이지를 벗어나면 다시 볼 수 없으니 바로 붙여넣기 하세요!
5
관리자 페이지에 입력
이 모달을 닫고 상단 GitHub Token 입력란에 붙여넣기 하면 자동 저장됩니다.
